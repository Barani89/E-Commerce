# NovaCart E-Commerce Frontend

This project is a modern React-based e-commerce storefront built with Vite, React Router, Tailwind CSS, Axios, and React Icons.

## Run locally

1. Install dependencies:
   npm install
2. Start the development server:
   npm run dev

## Included pages

- Home
- Products
- Product Details
- Cart
- Wishlist
- Login
- Register
- Profile
- 404 Not Found
